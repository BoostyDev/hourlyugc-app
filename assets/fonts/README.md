# Fonts from Figma

## Required Fonts:
1. **Switzer** - Used for headings and labels
   - Weights: Regular (400), Medium (500), SemiBold (600)
   
2. **DM Sans** - Used for body text
   - Weights: Regular (400), Medium (500)

## Download Sources:
- Switzer: https://fonts.google.com/specimen/Switzer (or custom)
- DM Sans: https://fonts.google.com/specimen/DM+Sans

Note: These fonts should match exactly what's used in Figma design.
